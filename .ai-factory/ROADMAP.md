# Project Roadmap

> План рефакторинга и развития Notion-like редактора (Vue 3 + TypeScript + Tiptap v3).
> Пункты расположены в логическом порядке выполнения: инфраструктура качества →
> типизация → тестовый фундамент → архитектура → Vue-практики → производительность →
> локализация → качество кода → технический долг → интеграция с бэкендом.
>
> Принятые продуктовые решения (2026-07-05):
> - **AI**: платное расширение Tiptap Pro AI не используется; в перспективе — собственное AI-расширение (Этап 9).
> - **Коллаборация**: внедряется позднее, после написания собственной бэкенд-части (Node.js).
> - **Хранение**: документы хранятся в MySQL; Node.js-бэкенд будет отдавать JSON-представление страницы (реализация обдумывается).
> - **Вставка таблиц**: из Google Sheets — работает; Excel — не проверено (нет установленного Excel).
